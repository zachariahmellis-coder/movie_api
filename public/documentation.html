Max's Movie API

This API provides movie, genre, director, and user endpoints for the myFlix application.

Getting Started

  1. Start the server: node index.js
  2. Home: http://localhost:8080
  3. Docs: http://localhost:8080/documentation

Endpoint Details

GET /movies

  • Description: Returns a list of all movies.
  • URL: /movies
  • Method: GET
  • URL Params: None
  • Request Body: None
  • Response Format: JSON array (or text confirmation)

GET /movies/:title

  • Description: Returns data about a single movie by title.
  • URL: /movies/:title
  • Method: GET
  • URL Params: title (string)
  • Request Body: None
  • Response Format: JSON object (or text confirmation)

GET /genres/:name

  • Description: Returns data about a genre by name.
  • URL: /genres/:name
  • Method: GET
  • URL Params: name (string)
  • Request Body: None
  • Response Format: Text or JSON describing the genre

GET /directors/:name

  • Description: Returns data about a director by name.
  • URL: /directors/:name
  • Method: GET
  • URL Params: name (string)
  • Request Body: None
  • Response Format: Text or JSON with director data

POST /users

  • Description: Registers a new user.
  • URL: /users
  • Method: POST
  • URL Params: None
  • Request Body Example: 
    {
  "Username": "MaxTestVersion2",
  "Password": "SuperSecret123",
  "Email": "maxtest@example.com",
  "Birthday": "1985-01-01"
}
  • Response Format: Text confirmation or JSON of created user

PUT /users/:username

  • Description: Updates user information for a given username.
  • URL: /users/:username
  • Method: PUT
  • URL Params: username (string)
  • Request Body Example: 
    {
  "Username": "MaxTestV2"
}
  • Response Format: Text confirmation or JSON of updated user

POST /users/:username/movies/:movieId

  • Description: Adds a movie to a user’s favorites list.
  • URL: /users/:username/movies/:movieId
  • Method: POST
  • URL Params: username, movieId
  • Request Body: None
  • Response Format: Text confirmation

DELETE /users/:username/movies/:movieId

  • Description: Removes a movie from a user’s favorites list.
  • URL: /users/:username/movies/:movieId
  • Method: DELETE
  • URL Params: username, movieId
  • Request Body: None
  • Response Format: Text confirmation

DELETE /users/:username

  • Description: Deregisters (deletes) a user.
  • URL: /users/:username
  • Method: DELETE
  • URL Params: username
  • Request Body: None
  • Response Format: Text confirmation
